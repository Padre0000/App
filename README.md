# تطبيق التجارة الإلكترونية

تطبيق تجارة إلكترونية متكامل يوفر منصة شاملة للمتاجر والمنتجات والطلبات.

## 🎯 المميزات الرئيسية

### للمستخدمين:
- ✅ تسجيل دخول سهل (بريد، هاتف، Google)
- ✅ صفحة رئيسية بتصميم جذاب
- ✅ عروض متحركة ومنتجات
- ✅ البحث والتصفية
- ✅ نظام الطلبات المتقدم
- ✅ خيارات التوصيل والاستلام

### لأصحاب المتاجر:
- ✅ إدارة متجر متكاملة
- ✅ إدارة المنتجات والمخزون
- ✅ تتبع الطلبات
- ✅ الإحصائيات والتقارير

### للأدمن:
- ✅ لوحة تحكم متقدمة
- ✅ إدارة المتاجر والموظفين
- ✅ إدارة العروض والمنتجات
- ✅ عرض جميع البيانات

## 🚀 البدء السريع

### Backend:
```bash
cd backend
npm install
npm start
```

### Frontend:
```bash
cd frontend
npm install
npm start
```

## 📊 البيانات المدعومة

- **دينار جزائري (DZD)** كعملة افتراضية
- **عربي (RTL)** كاتجاه افتراضي
- دعم كامل للعربية والإنجليزية

## 🔐 الأمان

- تشفير كلمات المرور
- JWT للمصادقة
- التحقق من الصلاحيات
- CORS محمي
